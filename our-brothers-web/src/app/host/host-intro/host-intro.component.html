מפגשים
אני רוצה לארח

מארחים יקרים, אנו שמחים שאתם מעוניינים לקחת חלק במפגשי 'האחים שלנו', לפתוח את הלב ואת הבית ולהוות חלק משמעותי במעגל הזיכרון וההנצחה.

ניתן לפתוח את המפגש לקהל הרחב, בתיאום מולך, או לסגור אותו למוזמנים בלבד. לתשומת ליבך, בדרך כלל, כמות הנרשמים לאירוח גדולה מכמות האחים הנרשמים, מכיון שלרוב קל יותר לפתוח את הבית מאשר להוביל ולשתף בסיפור אישי. אנחנו פה בשבילכם.
